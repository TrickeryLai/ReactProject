基于React+Webpack+ES6+router+redux的前端项目,基于antd前端框架,搭建了一些reactDemo,start命令配置了webpack-dev-server --devtool --process .

Steps

##INSTALL

.....npm install....

##DEVELOPMENT

....npm start....
....open http://localhost:8888....

##PROJECT

....npm build....
